# 交互式场景示例：COVID-19 数据探索 ## 场景 创建一个交互式 COVID-19 数据探索页面，使用 D3.js。 包含：时间序列折线图 + 可选择国家的交互。 ## 美学决策 - 主题：深色科学风格（Viridis 色板） - 定制："深色主题，用 Viridis 色板，字体大一点" ## 实现代码 ```html COVID-19 Cases Explorer

COVID-19 Daily New Cases

7-day rolling average · Select countries to compare

Source: Johns Hopkins University CSSE · Data as of 2024-01-01

``` ## 定制说明 本示例展示了自然语言定制 "深色主题，用 Viridis 色板" 的具体实现： - 背景色 → #1A1A1A（页面）/ #1E1E1E（图表区） - 文字色 → #E8E8E8（主文字）/ #999999（次要文字） - 网格色 → #333333 - 色板 → Viridis 多色相顺序色板 - 线条加粗至 2.5px（深色背景下需要稍粗）
