# 面积图（Area Chart）

## 适用场景
- 展示数据量随时间的变化趋势
- 强调"总量"概念
- 最佳系列数：1-3 个（非堆叠），2-5 个（堆叠）

## 数据格式
同折线图

## 美学参数
```json
{
  "area": {
    "opacity": 0.15,
    "stacked_opacity": 0.7,
    "gradient_top_opacity": 0.3,
    "gradient_bottom_opacity": 0.0
  }
}
```

## 设计要点

1. **单一面积图**：填充 opacity 0.15，叠加折线 2px
2. **堆叠面积图**：不设置透明度，各层使用实色填充以展示结构
3. **重叠面积图**：最多 3 个系列，使用低透明度（0.1-0.15）避免遮挡
4. **Y 轴**：堆叠时从 0 开始；非堆叠时可以不从 0 开始
5. **渐变填充**：优先使用从上到下的透明度渐变（顶部 0.3 → 底部 0.0）
