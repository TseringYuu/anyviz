# 雷达图（Radar Chart）

## 适用场景
- 多维度（3-10 维）的横向对比
- 展示多个实体在各维度上的表现差异

## 数据格式
```json
{
  "dimensions": ["速度", "准确性", "覆盖率", "稳定性", "易用性"],
  "entities": [
    { "name": "方案A", "values": [85, 72, 90, 78, 88] },
    { "name": "方案B", "values": [70, 88, 75, 92, 80] }
  ]
}
```

## 设计要点

1. **维度限制**：3-10 个维度（超过 10 个维度改用平行坐标图或柱状图）
2. **值域**：所有维度应共享相同的值域（如 0-100），否则需标准化
3. **实体数量**：≤ 5 个实体（超过时改用小倍数雷达图）
4. **填充透明度**：实体 > 2 时，使用半透明填充（opacity 0.15）而非实色
5. **网格线**：圆形网格线，每 20-25% 一条，颜色 #E0E0E0
